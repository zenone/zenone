# Steve Zenone

Python developer. Mac utilities, music production software, and AI experiments.

Most of my serious work lives in private repos: an AI agent framework and an algorithmic trading system. Here's what's public:

- **[crate](https://github.com/zenone/crate)** — DJ library tool: batch renaming, LUFS normalization, cue detection, Rekordbox export
- **[upkeep](https://github.com/zenone/upkeep)** — Mac health toolkit with web dashboard and automated maintenance
- **[powerflow](https://github.com/zenone/powerflow)** — Syncs Pocket AI voice recordings to Notion with smart formatting
- **[lastpass-to-apple-passwords](https://github.com/zenone/lastpass-to-apple-passwords)** — Migrate LastPass exports to Apple Passwords
